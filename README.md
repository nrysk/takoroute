# TakoRoute

たこ焼き検索アプリ

## 開発環境



- `.env.local`にホットペッパーグルメAPIのキーを設定

  ```.env.local
  HOTPEPPER_API_KEY=XXXXXXX
  ```

## プロジェクト構成

```
takoroute
|-- app
|   |-- _components # 汎用コンポーネント
|   |   |-- ui # 汎用UIコンポーネント
|   |-- _features # 固有機能
|   |-- _types # 共通の型
|   |-- _lib
```
